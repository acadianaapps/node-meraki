meraki/dashboard

Create a Meraki Dashboard API wrapper. This is an experimental feature and must be used at your own risk.

Source:
  • dashboard/index.js, line 5
Example
const target = 'n12'
const baseUrl = 'https://api.meraki.com' (default)
const email = 'john.doe@email.com'
const password = 'password1234'

const dashboardAPI = require('./lib/dashboard')({ target, baseUrl, email, password })
Parameters:
Name Type Attributes Default Description
target string <optional>
 'api'

The Meraki target
baseUrl string <optional>
 'https://api.meraki.com'

The Meraki base url
email string

The Meraki dashboard account email
password string

The Meraki dashboard account password
keepAlive boolean <optional>
 true

Whether to trigger the idleKeepAlive requests for the stored sessions
Returns:

The initialized Meraki Dashboard API wrapper
